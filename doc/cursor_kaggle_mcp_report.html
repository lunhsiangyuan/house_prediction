🎯 Cursor IDE MCP 概述

Model Context Protocol (MCP) 讓 Cursor IDE 能夠連接到外部工具和資料來源。透過 MCP，您可以將 Kaggle 的強大功能整合到您的編碼環境中。

Cursor MCP 優勢：

  • 直接在 IDE 中存取 Kaggle datasets
  • 瀏覽和分析 Kaggle competitions
  • 搜尋和參考 Kaggle kernels
  • 無需切換應用程式即可使用 Kaggle 資源

⚡ 效能優化建議

最佳化您的 MCP 體驗：

  • 啟用/禁用控制：在 Cursor Settings 中可以啟用或禁用不需要的 MCP servers
  • 專案級別配置：為特定專案建立 `.cursor/mcp.json` 避免工具過載
  • 環境變數：使用 Cursor UI 設定環境變數，避免在配置檔案中暴露敏感資訊
  • 工具分類：根據任務類型選擇性啟用相關 MCP servers

🔧 故障排除

常見問題解決：

  • MCP Server 未啟動：檢查 `kaggle-mcp` 命令是否在 PATH 中
  • 認證失敗：確保 `~/.kaggle/kaggle.json` 有正確的檔案權限 (600)
  • 配置更新未生效：重新啟動 Cursor IDE
  • 工具未顯示：檢查 Cursor MCP Settings 中是否啟用了 Kaggle server
  • 環境變數：可在 MCP Settings 中設定 `env` 變數
